# Discord Gaming Bot & Wiki Integrator

An advanced Discord chatbot built with Python that implements real-time web scraping and large language model (LLM) integration to provide intelligent, context-aware gaming strategies for V Rising.

## Features
- **Real-time Web Scraping:** Dynamically pulls the latest data, boss mechanics, and items from the official Fandom wiki.
- **RAG Architecture Context Layer:** Feeds parsed wiki data directly into the system instructions to prevent LLM hallucinations.
- **Modern SDK Integration:** Powered by Google's latest official `google-genai` SDK and the `gemini-2.5-flash` model.
- **Asynchronous & Robust:** Built using `discord.py`'s async capabilities with custom edge-case handling for Discord's 2000-character message limits.

## Tech Stack
- **Language:** Python 3.14
- **AI Framework:** Google Gemini API (`google-genai`)
- **Scraping:** BeautifulSoup4 & Requests
- **API Layer:** discord.py
- **Environment Management:** python-dotenv (Secure API keys management)

## Repository Structure
- `v_rising_ai.py` - Core Discord bot logic, asynchronous message processing, and Gemini API integration.
- `wiki_kaziyici.py` - Custom web scraper that handles web requests, session management, and parsing logic.
- `v_rising_bot_hafizasi.txt` - Local structured database generated by the scraper to act as the grounding context for the AI.
